# First thing to do

To install node modules dependencies:

- open the VSCode terminal:

```
Terminal > New Terminal > Choose folder
```

- type in the VSCode terminal :
  
```
npm install 
```

# How to launch node.js Server

Type in the terminal npm start, easy.
